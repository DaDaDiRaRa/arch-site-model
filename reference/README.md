## ngii_layer_codes.{csv,json}

NGII(국토지리정보원) 수치지형도Ver2.0 레이어 분류 코드 674개 전체 (`code`, `category`, `name`).
공개 표준 코드 체계 — 저작권 문제 없음.

**출처**: KBS TopoMap 3.9.2(SketchUp 확장) 배포판에 포함된 `codetag.txt`(코드) +
`codename.txt`(이름) 두 파일을 라인 순서로 매칭해 재구성. `category`는 코드 첫 글자로 유도
(A=교통, B=건물, C=시설물, D=식생·경작, E=수계, F=지형/등고선, G=경계, H=주기·문자).

우리 `contour_bake.py`는 지금 코드가 아니라 필드명("등고수치"/"수치")으로 직접 파싱하므로
이 표는 당장 쓰이지 않음.
